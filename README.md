# 島開拓ゲーム v2

走り回って資源を集め、現地へ運び、建物と新しいエリアを完成させるスマホ向けカジュアルゲームです。HTML / CSS / JavaScript / Canvas とローカルPNG素材だけで動作します。

## v2の主な追加要素

- 5つの段階解放エリア：はじまりの浜辺、石の海岸、緑の草原、鉱山エリア、港エリア
- 建築予定地へ近づくと、手持ち素材を少しずつ自動投入する現地建築
- 木材、石材、食料、繊維、鉱石の段階解放
- 港完成まで続く17段階のクエスト
- 現在の目的地を示すマーカーと画面端の矢印
- 木へ歩き、伐採し、小屋へ木材を運ぶ作業員
- 住宅や畑の周辺を歩く住民
- 建築完成、新エリア解放、売却、クエスト達成時の演出強化

## 遊び方

- 左下のジョイスティックで移動します。PCでは矢印キー / WASDも使用できます。
- 木や解放済みの資源へ近づくと自動で採取します。
- 売却所へ資源を運び、右下の売却ボタンでコインに換えます。
- 建築予定地へ素材を持って近づくと、自動で投入されます。
- 右上のクエストと黄色い目的地ガイドを頼りに港完成を目指します。
- 開拓メニューでは建築進捗の確認、場所の表示、作業員の雇用ができます。

## セーブデータ

進行状況は `localStorage` に自動保存されます。

- key: `island_dev_game_save`
- save version: `2`
- `migrateSave(save)` で旧形式を安全なv2構造へ変換します。
- ゲーム内のリセットボタンから、確認後に最初から開始できます。

## GitHub Pagesで公開

リポジトリ直下に以下を同じ構成で配置してください。

```text
index.html
style.css
main.js
README.md
assets/
```

ビルド、サーバー、外部CDN、外部ライブラリは不要です。GitHub Pagesの公開元をリポジトリのルートに設定すれば、そのまま動作します。

## iPhone Safari

縦画面とセーフエリアに対応しています。ページのスクロール、選択、ピンチズームがゲーム操作中に暴発しないよう設定しています。ホーム画面追加用のメタ情報も含まれています。
